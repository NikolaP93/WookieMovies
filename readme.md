### installing

npm/yarn install

### install cocoapods

npx pod-install with npx
or
cd ios
pod install

### compiling

tsc -p . --skipLibCheck

### run on android

npm/yarn run android

### run on ios

npm/yarn run ios

### run tests

yarn test

### extra features

User can set a favorite movie while on the Details screen via the favorite button;
